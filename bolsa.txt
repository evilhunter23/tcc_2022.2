Inicialmente é escolhida uma ação cujos dados se deseja obter. Se faz necessário digitar o nome da ação, composto geralmente por 4 letras e um número. Para se obter dados de empresas brasileiras é preciso adicionar o sufixo “.sa” no final do nome. Como exemplo, para uma ação da Petrobrás temos “petr4.sa”. Para fazer a predição dos valores, primeiro será necessário obter os dados da bolsa de valores, de uma respectiva data até outra data, e organizá-los em um dataset. Estes serão os dados utilizados para treinar o modelo de predição, obtidos a partir do Yahoo Finanças. O programa se inicia definindo a data do dia atual, e obterá os dados de uma certa quantidade de anos anteriores, até hoje.
